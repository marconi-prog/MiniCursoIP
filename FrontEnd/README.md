# FrontEnd

Esta pasta contém exemplos de **desenvolvimento frontend** utilizados no minicurso.

Aqui estão os arquivos responsáveis pela interface das aplicações web.

## Estrutura

```
FrontEnd
├ html
├ css
└ js
```

### HTML

Arquivos responsáveis pela **estrutura da página**.

Exemplos de conteúdo:

* criação de páginas
* formulários
* estrutura de layout

### CSS

Arquivos responsáveis pelo **estilo das páginas**.

Exemplos de conteúdo:

* cores
* layout
* responsividade
* estilização de componentes

### JavaScript

Arquivos responsáveis pela **interatividade das páginas**.

Exemplos de conteúdo:

* manipulação do DOM
* eventos
* comunicação com APIs

## Objetivo

Apresentar os conceitos básicos de desenvolvimento frontend e sua integração com aplicações backend.
