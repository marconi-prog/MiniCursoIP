# Backend

Esta pasta contém os exemplos e projetos de **backend** utilizados no minicurso.

Aqui são apresentados dois frameworks populares do ecossistema Python:

## Estrutura

```
Backend
├ app-Flask
└ django
```

### Flask

A pasta `app-Flask` contém exemplos de aplicações utilizando **Flask**, um microframework Python para desenvolvimento web.

Principais características:

* aplicações simples
* fácil aprendizado
* ideal para introdução ao desenvolvimento web

### Django

A pasta `django` contém exemplos de aplicações utilizando **Django**, um framework web completo para Python.

Principais características:

* arquitetura MVC (Model-View-Template)
* sistema de ORM integrado
* painel administrativo automático

## Objetivo

O objetivo desta parte do minicurso é apresentar conceitos básicos de:

* desenvolvimento web
* APIs
* arquitetura de aplicações web
* integração com banco de dados
