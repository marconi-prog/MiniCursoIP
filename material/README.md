# Material de Apoio

Esta pasta contém materiais complementares utilizados durante o minicurso.

Os arquivos presentes aqui servem como **apoio teórico ou prático** para os conteúdos abordados nas aulas.

## Conteúdo

Podem incluir:

* PDFs de apoio
* artigos
* materiais de referência
* recursos externos utilizados durante as aulas

## Objetivo

Fornecer materiais adicionais para reforçar o aprendizado e permitir que os alunos explorem os temas estudados com mais profundidade.
